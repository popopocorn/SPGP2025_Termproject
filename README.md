※GameObject※

Mushmom - 플레이어
hp, ad, as를 가진 플레이어 객체, 조이스틱의 pow, angle정보를 받아 해당방향으로 이동, 공격을 함
경험치 객체, 적 객체와 상호작용 구현 예정

bullet - 공격
플레이어가 발사하는 객체, 플레이어와 동일하게 조이스틱에 angle에 따라 발사 방향이 정해짐
적 객체와 충돌체크 구현 예정
7, 8주차에는 플레이어의 강화 요소에 따라 발사체를 다르게 변경할 예정
